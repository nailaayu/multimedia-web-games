# MULTIMEDIA WEB GAMES

Project game multimedia berbasis web menggunakan HTML, CSS, dan JavaScript.

## Fitur

* Game shooter dengan gameplay interaktif
* Game matematika dengan sistem skor
* Game klik hewan
* Tampilan responsif
* Efek suara dan animasi sederhana
* Desain game berbasis web

## Teknologi

* HTML5
* CSS3
* JavaScript

## Cara Menjalankan

```bash
Buka file index.html menggunakan browser
```
